# OpenICF 2.0.1 Released
[Download](https://github.com/OpenIdentityPlatform/OpenICF/releases/tag/2.0.1)

## What's new
* Updated target JDK to 11 and migrated to JakartaEE 9 for improved compatibility with modern Java environments
* Added support for LTS JDK 25, ensuring long-term stability and security
* Updated base Docker image to Java version 25 LTS for containerized deployments
* Updated OpenDJ dependency to version 5.0.1 with latest improvements

Full changeset ([more details](https://github.com/OpenIdentityPlatform/OpenICF/compare/1.8.0...2.0.1))
